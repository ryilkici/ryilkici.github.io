Pymun

Pymun is a practical random number generator. Any user can generate numbers with Normal, Binomial, Uniform, or Poisson distributions and user can download the numbers data in a .csv file.

Pymun has developed with Python. Back-end of the Pymun has developed with Flask web framework package, and front-end of the Pymun has developed with HTML5, CSS3, JavaScript and Bootstrap 4. Pymun can generate numbers with 5 different method just for now. It will be better soon!
